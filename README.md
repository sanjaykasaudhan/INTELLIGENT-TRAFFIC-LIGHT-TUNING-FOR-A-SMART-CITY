# INTELLIGENT-TRAFFIC-LIGHT-TUNING-FOR-A-SMART-CITY
Developed an AI-powered traffic signal optimization system using Genetic Algorithms to intelligently manage a 12-junction urban network, significantly reducing congestion and improving traffic flow efficiency.
